A Literature Complete Catalog

The Multi-AGN Catalog represents a foundational archive for multi-AGN science.

Note: You are currently viewing the beta version of this website and catalog. Updates are on the way.

Designed to bring uniformity and structure to the field, this catalog (colloquially referred to as 'The Big MAC') encompasses all confirmed and candidate dual AGN (separations 100 kpc to 0.1 pc, and including binary quasars), (gravitationally bound) binary AGNs, and recoiling AGNs (post coalescence). Designed for literature completeness, this catalog also includes objects that may well turn out to be lenses or other phenomenon.

Dual and Multi-AGNs

This library defines dual (and multi-) AGNs as having projected separations of ~0.1<rp<100 kpc .

  • See More

Binary AGNs

This library defines binary AGNs as having separations of less than 0.1 kpc (or less than 100 pc).

  • See More

Recoiling AGN Candidates

This library includes all AGNs flagged as recoil candidates, which would be the result of a binary SMBH merger event.

A Living Library

Do you have suggestions for the library?

The Multi-AGN Library is intended to be a living-library, with regular updates pushed to the GitHub repositories throughout the year and full annual releases.
But this catalog would not be possible without contributions from researchers like you!
Please use the buttons below to contact us if you would like to submit new objects to the catalog, new information for current targets included in the catalog, if you would to contribute to the development and curation of this catalog, or if you simply have suggestions for future releases.
